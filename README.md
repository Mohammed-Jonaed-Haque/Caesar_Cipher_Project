# Caesar Cipher 🔐

## Overview

This project is a  **Caesar Cipher program written in Python** that allows users to encrypt and decrypt messages by shifting letters in the alphabet by a specified number.

---

## Features

* Encode (encrypt) messages
* Decode (decrypt) messages
* Custom shift values
* Supports spaces and special characters
* Allows users to run the program multiple times

---

## Technologies Used

* Python
* Basic programming concepts:

  * Functions
  * Loops
  * Lists
  * Conditional statements

---

## How It Works

1. The user chooses whether to **encode** or **decode** a message.
2. The user enters the message they want to process.
3. The user enters a **shift number**.
4. Each letter in the message is shifted through the alphabet.
5. The program displays the encrypted or decrypted result.

Example:

```
Message: hello
Shift: 3

Encoded result: khoor
```
